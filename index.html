Step 1: Access Steam

Open your Steam account.

  • Go to the Library section.
  • In the left column, ensure the following options are selected:
    • Games
    • Tools

Step 2: Install Dedicated Server

Search and install the "Last Oasis" dedicated server.

  • If you have a Windows server with root access, install Steam on the machine and begin installation.

Step 3: Locate `MistServer.exe` File

After installation completes, navigate to the installation directory containing `MistServer.exe`.

Typical path:

Steam\LastOasis\steamapps\common\Last Oasis - Dedicated Server

Step 4: Create a `.bat` File

Create a text file named `Server1.bat` in the same directory as `MistServer.exe`.

  • You can create additional servers by making files like `Server2.bat`, `Server3.bat`, etc.

Step 5: Configure File Explorer Settings

Open the Control Panel and navigate to File Explorer Options.

  • Disable the option Hide extensions for known file types.

Step 6: Change File Extension from `.txt` to `.bat`

Rename the `Server1.txt` file to `Server1.bat` by removing the `.txt` extension.

Step 7: Log in to Server Control Panel

Log in to the Last Oasis server control panel using your Steam account:

  • URL: http://myrealm.lastoasis.gg/Account/Signin
  • Navigate to the Settings tab.

Step 8: Generate New Provider Key

Create a new Provider Key. The Customer Key cannot be changed.

  • Follow the instructions to generate the key.

Step 9: Copy Information from Official Documentation

Copy configuration information from the official documentation file for the `.bat` file and paste it into your created file (`Server1.bat`).

Step 10: Configure External IP Address

Determine your external IP address using websites like:

  • https://whatismyipaddress.com/de/meine-ip
Note: If hosting from home, your IP may change every 24 hours or when restarting your router. Update the IP address in the `.bat` file as needed.

Step 11: Advanced `.bat` File Configuration

You can create a more structured `.bat` file for better visibility and organization. 


@echo off
:: #########################################################
:: # Set Variables #
:: #########################################################

set Identifier=neon_server1
set Port=5555
set CustomerKey=a54iesJ4Ga0MC1Pt
set ProviderKey=AM0hLmhwg75xIBdH
set Slots=5
set QueryPort=27015
set OCA=YOUREXTERNALIP

MistServer.exe -log -identifier=%Identifier% -port=%Port% -CustomerKey=%CustomerKey% -ProviderKey=%ProviderKey% -slots=%Slots% -QueryPort=%QueryPort% -OverrideConnectionAddress=%OCA%
exit

Step 12: Realm Management

You can have more than 2 active servers running simultaneously. Each server represents an Oasis in the Realm.

  • To manage multiple servers:
    • Change ports and QueryPorts to avoid conflicts.
    • Example port ranges:
      • Ports: 29015-29017
      • Query Ports: 5565-5567
    • The identifier can be the same name or different for each server.
    • View server configurations in the dashboard under Game Servers.

Realm Configuration:

  • After creating a Realm, click View to display its settings.
  • You can rename the Realm for in-game display, adjust multipliers, manage admins, and more.
  • Additional options will be available in future updates.

Step 13: Oasis Management

You can manage multiple Oases for servers. Only 3 Oases will be active when starting 3 servers.

  • To launch event maps in different root systems or local systems:
    • The Provider Key and Customer Key must match those used in the same Realm.
    • Event maps can be enabled under the Events section in the dashboard.
